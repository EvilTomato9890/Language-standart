# DIV

Делит одно число на другое

---
## Свойства и функционал 

### Функционал ASM
Делит 2-ое число со стека на 1-ое со стека и бросает результат на стэк.

>**Вид в ASM**
> ```
> PUSH 1
> PUSH 2
> DIV
> ```

### Свойства и особенности
- Деление осуществляется согласно стандарту IEEE 754
- Согласно стандарту результатом может являться +-0, +-inf, NaN
---

## Формат узла

> [!info] Обозначения
> **...** — нет ограничений.

| Ребенок | Имя | Класс | Примечание |
|:-----:|:---:|:-----:|:----------:|
| left  | ... | Expression |  Делимое  |
| right | ... | Expression |  Делитель  |
**Иные требования к детям:** Отсутствуют

---

### Примеры

> **Минимальныe примеры:**
> ```
> DIV (1 () ()) (1 () ())
> ```
> *Результат: 1*


> **Стандартныe примеры:**
> ```
> DIV (1 () ()) 
> 	(PLUS (1 () ()) 
> 		  (1 () ()))
> ```
> *Результат: ~0.5*

> **Пограничныe примеры:**
> ```
>DIV (1 () ()) (0 () ())
> ```
> *Результат: +inf*
> <br>
> ```
>DIV (-1 () ()) (0 () ())
> ```
> *Результат: -inf*
> <br>
> ```
>DIV (0 () ()) (-1 () ())
> ```
> *Результат: -0*
> <br>
> ```
>DIV (0 () ()) (1 () ())
> ```
> *Результат: 0*
> <br>
> ```
>DIV (0 () ()) (0 () ())
> ```
> *Результат: NaN*
