# MINUS

Вычитает из одного числа другое.

---
## Свойства и функционал 

### Функционал ASM
Вычитает из 2-ого числа со стека 1-ое число со стека и бросает результат на стэк

>**Вид в ASM**
> ```
> PUSH 1
> PUSH 1
> SUB
> ```
### Свойства и особенности
Отсутствуют

### Примеры

> **Минимальный пример:**
> ```
> MINUS (0 () ()) (0 () ())
> ```
> *Результат: 0*


> **Стандартный пример:**
> ```
> MINUS (MINUS (1 () ())
> 			(-1) () ())
> 	  (1 () ())
> ```
