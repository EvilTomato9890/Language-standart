# AND

Производит логическое И

---
## Свойства и функционал 

### Функционал ASM
1. Если какой-либо из 2 чисел со стека это NaN, то кидает на стек NaN
2. Если какой-либо из этих 2-х чисел это +-0, то кидает на стек 0, иначе кидает 1

>**Вид в ASM**
> ```
> PUSH 0
> PUSH 1
> AND
> ```

### Свойства и особенности
Отсутствует

### Примеры

> **Минимальныe примеры:**
> ```
> AND (0 () ()) (1 () ())
> ```
> *Результат: 0*


> **Стандартныe примеры:**
> ```
> AND (EQ (1 () ()) 
> 		(1 () ()))
> 	 (1 () ())
> ```
> *Результат: 1*
> <br>
> ```
> AND (0.1 () ())
> 	(1 () ())
> ```
> *Результат: 1*

> **Пограничныe примеры:**
> ```
> AND (NaN () ()) (1 () ())
> ```
> *Результат: NaN*
