# IF

Проверяет условие, если не выполняется, то пропускает часть кода

---
## Свойства и функционал 

### Функционал ASM
1. Берет результат условия со стека. 
2. Проводит [[EQ]] вместе с 1, но результат на стек не кладет. 
3. Если результат — 1, то ничего не делает
4. Если результат — 0, то переходит по соответствующей метке

>[!note] Примечание
> В ASM отсутствует реализация одной инструкцией

>**Вид в ASM**
> ```
> PUSH 1
> PUSH 1
> EQ
> PUSH 1
> JNE :if_false
> PUSH 1
> OUT
> :if_false
> ```

> [!info] Обозначения
> **...** — нет ограничений.

| child | имя | класс | группа| примечание |
|:-----:|:---:|:-----:|:-----:|:--------:|
| left  | ... | Expression | ... | Условие |
| right | ... | ... | ... | При выполнении условия |

**Иные требования к детям:** Отсутсвуют

---

### Примеры

> **Минимальныe примеры:**
> ```
> IF  (1 () ()) 
> 	()
> ```
> *Результат: 1 на стеке*

> **Стандартныe примеры:**
> ```
> IF (EQ  (1 () ())
> 		(1 () ())
> 	PRINT (1 () ()))
> ```
> *Результат: 1 в выходном потоке*

> **Пограничныe примеры:**
> ```
> Отсутствуют
> ```
