# RETURN

Завершает функцию и возвращает переданное значение

---
## Свойства и функционал 

### Функционал ASM
Записывает возвращаемое значение на стек и вызывает ret

>**Вид в ASM**
> ```
> foo:
> PUSH 1
> RET
> ```

### Свойства и особенности
- Не может существовать внутри тела процедуры ([[PROC]])

---

## Формат узла

> [!info] Обозначения
> **...** — нет ограничений.

| Ребенок | Имя | Класс |  Примечание |
|:-----:|:---:|:-----:|:--------:|
| left  | ... | Void | ... |
| right | ... | Expression | Возвращаемое значение |

**Иные требования к детям:** Отсутствуют

---

### Примеры

> **Минимальныe примеры:**
> ```
> RETURN () (1 () ())
> ```
> *Результат: возврат 1*


> **Стандартныe примеры:**
> ```
> RETURN () (PLUS (1 () ())
> 				(1 () ()))
> ```
> *Результат: возврат 2*

> **Пограничныe примеры:**
> ```
> Отсутствуют
> ```
